# MetaSignal v1.0.0

## Release Summary

MetaSignal v1.0.0 is the first public showcase release of the production-simulated experimentation, metrics intelligence, and streaming observability platform.

This release packages the core repo evidence for:

- metric governance
- denominator conflict detection
- data quality gates
- deterministic experiment assignment
- SRM validation
- CUPED experiment readouts
- CUPED A/A validation
- guardrail-first decisioning
- right-censoring for delayed guardrails
- anomaly detection
- golden scenario validation
- FastAPI evidence retrieval
- operational history simulation
- streaming early-warning extension
- stream-batch reconciliation
- GitHub Pages evidence dashboard

## Claim Boundary

MetaSignal is solo-built, non-production, and production-simulated.

It does not claim real production deployment, real company users, real production traffic, Kafka/Flink production infrastructure, real analyst decisions, or a real A/B treatment effect from RetailRocket.

Streaming alerts are provisional investigation signals. Batch remains authoritative.
